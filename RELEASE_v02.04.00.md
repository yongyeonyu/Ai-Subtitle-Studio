# RELEASE v02.04.00

## 핵심 변경
- PHASE1-C 기능 보존형 Apple 스타일 UI 개선을 적용했습니다.
- 전역 메뉴바를 `ui/menu_bar.py`로 분리하고, 중복 메뉴를 정리했습니다.
- 불필요해진 전역 `이전` / `다음` 액션을 제거했습니다.
- 창 폭이 절반 이하로 줄면 메뉴 버튼 텍스트를 숨기고 아이콘만 표시합니다.
- 상단 상태 레일에서 생성/편집/저장/출력/종료 상태를 표시합니다.
- 왼쪽 사이드바에 iCloud/NAS 자동처리 상태와 프로젝트/영상/자막 정보 패널을 복구했습니다.
- 빈 작업 화면 중앙에 파일/폴더 빠른 열기 액션을 추가했습니다.

## 에디터 / 비디오 / 타임라인
- 비디오 프리뷰 subtitle overlay 갱신과 자막 에디터 자동 스크롤을 보강했습니다.
- 비디오 프리뷰 텍스트 라벨의 박스형 테두리를 제거했습니다.
- 비디오 프리뷰 저화질 프록시는 원본 fps를 유지하도록 정리했습니다.
- 16:9 / 1:1 편집 기준 비디오 폭 확장 토글 위치를 보정했습니다.
- 디테일 타임라인 다이아몬드 핸들을 작은 45도 다이아몬드 형태로 정리했습니다.
- 자막 세그먼트의 좌우 화살표, 무음 구간, 화자/음성 웨이브폼 affordance는 유지했습니다.

## 설정 / 공용 스타일
- `ui/style.py`의 공용 line icon 세트를 확장했습니다.
- 설정창 공통 버튼 높이, 최소 너비, radius를 정리했습니다.
- 설정 하단 버튼에 저장/취소/확인/초기화 아이콘을 공통 적용했습니다.
- 자막 간격 시뮬레이터의 색상, 블록, 도움말 표시를 다크 프로 톤으로 정리했습니다.
