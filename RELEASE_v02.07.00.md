# RELEASE v02.07.00

## 핵심 변경
- PHASE1-D STT 모드를 추가했습니다.
- STT ON 상태에서 시작 버튼을 누르면 Whisper/LLM 없이 최고 민감도 VAD-only 탐지로 텍스트 없는 STT 세그먼트를 생성합니다.
- STT 세그먼트는 프로젝트 파일에 저장하고, 완료 전에는 SRT 저장 대상에서 제외합니다.
- 마이크 one-shot STT 입력 경로를 추가해 사용자가 듣고 말한 내용을 현재 STT 세그먼트에 적용할 수 있게 했습니다.

## UI / 편집
- 상태 아이콘 레일을 상단 바에서 프로젝트 바 상단 2줄로 이동했습니다.
- 저장 상태 라벨을 프로젝트 바 하단으로 옮기고, 중앙 `AI Subtitle Studio` 제목 바를 제거했습니다.
- 상단 공간을 회수해 터미널 로그 영역을 키웠습니다.
- 저장 버튼 옆에 Undo/Redo 아이콘 버튼을 추가하고 에디터 undo/redo 라우팅에 연결했습니다.
- `자동설정` / `캐쉬삭제` 버튼은 한 줄 텍스트와 동일 높이로 정리했습니다.
- 파란 웨이브폼은 VAD/음성 감지 블록, 초록 웨이브폼은 원본 오디오 에너지로 구분했습니다.

## 화자 세그먼트
- 화자 학습 기능을 자막 세그먼트 우클릭 메뉴에서 화자 세그먼트 우클릭 메뉴로 이동했습니다.
- 화자 세그먼트 왼쪽 클릭 시 사용 중인 화자 목록을 색상 아이콘과 함께 표시하고, 선택한 화자로 변경합니다.
- 미인식 구간은 `홍길동`, 단일 인식 구간은 `화자1`, 복수 인식 구간은 `화자1 / 화자2` 형식으로 표시합니다.

## 안정화
- `기존 자막 사용`, `저장되지 않은 변경 사항`, STT 저장 확인창을 공용 Apple 스타일 메시지 박스로 통일했습니다.
- STT VAD-only 완료 후 추가 동작이 없어 메인으로 복귀하는 600초 대기 로그를 오류가 아닌 안내 문구로 변경했습니다.
- 사이드바에 이동한 상태 레일이 `show_home()` 재호출 때 삭제되지 않도록 재부착 방어를 추가했습니다.

## 문서
