# Planning Queue

This folder owns active plans, completed action-item archives, rejected ideas,
repeat-prevention lessons, and owner-review scratch ideas.

Canonical files:

- `ACTION_ITEMS.md`
- `COMPLETED_ACTION_ITEMS.md`
- `waste_action_item.md`
- `lesson_n_learned.md`
- `idea.md`
- `../HANDOFF.md`

Rules:

- Keep `ACTION_ITEMS.md` active-only.
- Move completed action-item summaries to `COMPLETED_ACTION_ITEMS.md`.
- Keep rejected experiments in `waste_action_item.md`.
- Keep repeat-prevention lessons in `lesson_n_learned.md`.
- Do not recreate root planning docs; root keeps `../../AGENTS.md` only.
