# Planning Queue

This folder is a navigation bucket for planning documents. It does not replace the root active queue.

Canonical files:

- `../../ACTION_ITEMS.md`
- `../../COMPLETED_ACTION_ITEMS.md`
- `../../waste_action_item.md`
- `../../lesson_n_learned.md`
- `../HANDOFF.md`

Rules:

- Keep `ACTION_ITEMS.md` active-only.
- Move completed action-item summaries to `COMPLETED_ACTION_ITEMS.md`.
- Keep rejected experiments in `waste_action_item.md` and repeat-prevention lessons in `lesson_n_learned.md`.
- Do not move the root queue files into this folder without explicit owner approval and automation compatibility proof.
