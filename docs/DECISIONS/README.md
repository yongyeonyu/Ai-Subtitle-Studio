# Decisions Log Guide

이 디렉터리는 세션 메모가 아니라, 여러 세션에 걸쳐 유지되어야 하는 저장소 수준의 결정만 기록하기 위한 자리입니다.

## When to add a decision note

아래와 같은 경우에만 새 결정 문서를 추가합니다.

- 구조 경계를 바꿔 여러 owner 파일에 영향을 줄 때
- 기본 검증 절차가 바뀔 때
- 릴리스 규칙이나 작업 규칙이 바뀔 때
- 다음 세션들이 반복해서 같은 판단을 하지 않도록 고정해야 할 때

## What a decision note should contain

- 결정 제목
- 날짜
- 배경
- 결정 내용
- 영향 범위
- 검증 또는 후속 문서 갱신 항목

## What not to store here

- 임시 계획
- 개인 scratch 메모
- 비공개 정보
- 단발성 버그 triage 기록

결정 문서가 필요 없다면 이 디렉터리는 비워 두어도 됩니다. 억지로 채우지 말고, 실제로 반복 참고 가치가 있는 변경에만 사용합니다.
