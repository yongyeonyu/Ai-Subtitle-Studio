================================================================================
  PHASE2 — AI 러프컷 편집
================================================================================

── UI / 기반 ──

  P2-U0  | UI 레이아웃 설계
         | - 대표님이 그림으로 레이아웃 전달
         | - Copilot이 PyQt6 뼈대 UI 생성 → 확인 → 기능 추가

  P2-U1  | 전체화면 비디오 프리뷰
         | - 러프컷 최종안을 풀스크린으로 미리보기

── 프로젝트 / 데이터 ──

  P2-P1  | 프로젝트 파일 확장
         | - AI 러프컷 정보 저장: 컷 순서 변경/삭제/챕터/주제/summary

  P2-P2  | 챕터 자동 생성
         | - AI가 전체 자막을 챕터로 묶고 주제 부여

  P2-P3  | 챕터 기반 summary
         | - 챕터 순서에 따른 전체 영상 요약 자동 생성

  P2-P4  | 사용자 피드백 학습
         | - 러프컷 의견을 md 파일로 저장
         | - LLM이 스타일 지속 학습 → 러프컷 고도화

── AI 러프컷 핵심 ──

  P2-A1  | 멀티 AI 러프컷 생성
         | - 사용자가 선택한 여러 AI 모델이 각각 러프컷 후보 생성

  P2-A2  | 러프컷 후보군 비교/선택
         | - 후보군 나란히 보고 최종 결정

  P2-A3  | 러프컷 길이 설정
         | - AI 자동추천 (전체 내용 보고 판단)
         | - 사용자 목표 시간 설정: 목표 ± 오차
         | - 최소 단위 1분 (변경 가능하나 1분 미만 불가)
         | - 예: 10분 타겟 → 9~11분 사이로 생성

  P2-A4  | 컷 순서 변경/삭제
         | - 사용자가 AI 제안을 수동 조정 가능

── 출력 ──

  P2-O1  | 무손실 러프컷 출력
         | - 영상 품질 손실 없는 출력
         | - 멀티카메라 시 최고 품질 + 최고 fps 통일

  P2-O2  | 러프컷 자막 자동 재배치
         | - 컷 순서 변경 시 프로젝트 자막만 변경
         | - 개별 영상 자막 원본은 유지

  P2-O3  | 러프컷 SRT + 자막 영상 출력
         | - Phase1 출력 기능 확장

  P2-O4  | FCPXML 출력
         | - Final Cut Pro에서 최종 편집 가능하도록 XML 내보내기

── 자막 스타일 학습 ──

  P2-L1  | 교정 diff 자동 저장
         | - 자막백업 최신 bak (AI 원본) vs 사용자 수정 srt (최종)
         | - diff를 corrections.json에 자동 누적

  P2-L2  | "자막학습" 메뉴
         | - 에디터 위젯 내 메뉴 신설
         | - "학습" 버튼으로 프롬프트/md/LoRA 갱신 실행

  P2-L3  | 학습 실행 (Phase2)
         | - 교정 로그 → 최근 N개 프롬프트 주입
         | - 100개마다 LLM이 패턴 분석 → auto_style_guide.md 재생성

── 화자 인식 확장 ──

  P2-S1  | 멀티 화자 학습
         | - 화자 1/2/3 모두 개별 학습 (현재 화자1만)

  P2-S2  | 영상 구간 선택 학습
         | - 편집 중 영상에서 구간 선택 → 해당 구간 오디오 추출
         | - 화자별 학습 WAV로 output/data에 저장

  P2-S3  | 학습 소스 이중화
         | - 기존: WAV 파일 불러오기
         | - 추가: 현재 영상에서 구간 발췌

── 자동화 설정 ──

  P2-AUTO1 | "경로설정" → "자동화 설정" 리네이밍
           | - 메인 화면 메뉴 변경

  P2-AUTO2 | 자동화 파이프라인 설정 UI
           | - 각 단계 ON/OFF + 세부 옵션

  P2-AUTO3 | 원스톱 순차 실행
           | - STT → 자막 → 러프컷 → 출력 → 업로드 자동 진행

  P2-AUTO4 | 탭 기반 통합 설정 패널
           | - 하나의 화면: STT/자막/화자/러프컷/출력/업로드 탭
           | - 에디터 진입 없이 메인에서 모든 설정 수정 가능

  P2-AUTO5 | 에디터 설정과 동기화
           | - 자동화 설정 변경 = 에디터 설정과 동일 데이터 공유

  P2-AUTO6 | 자동시작 → 전체 파이프라인 연결
           | - 자동시작 시 STT→러프컷→업로드까지 자동 진행
