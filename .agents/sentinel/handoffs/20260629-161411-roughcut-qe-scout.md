### G4 네거티브/실패 예외 시나리오 QE 검토 보고서 (서린)

#### 1. 시드 스위칭 시 작업 유실 (Seed switch overwrite)
- **실패 상황**: 사용자가 노드 드래그를 통해 정밀하게 조립해 둔 레이아웃 상태에서 실수로 시드 스위치 버튼을 눌렀을 때, 경고 창 및 임시 백업이 작동하지 않아 작업 데이터가 영구 소실되는 버그 검증.
- **검증 기준**: 수동 정렬 이력이 1회 이상 있는 상태에서 시드 전환 요청 시, 무조건 Modal Dialog 경고창이 팝업되고, '취소' 클릭 시 직전 2D 캔버스 상태가 픽셀 및 상태 정보 오차 없이 100% 유지되어야 합니다.

#### 2. 대체 촬영본 전환 시 싱크 오차 (Alternate takes drift)
- **실패 상황**: 대체 테이크(Alternate take)를 선택했을 때, 미세한 싱크 오차(ms 오프셋) 보정 실패로 자막 표시 시간과 실제 말소리가 일치하지 않아 립싱크가 완전히 무너지는 드래프트 현상 검증.
- **검증 기준**: VAD 파형 에너지 비교를 통해 보정 오프셋(offset) 값을 재계산하여 프리뷰 시점에 정확히 적용하는지, 비디오 탐색(Seek) 및 재생(Play) 속도와 연동하여 1프레임 미만 정밀도로 자막 표시 싱크가 고정되는지 검사합니다.

#### 3. 세그먼트 분할/병합 시 자막 오버랩 및 역전 (Split/merge overlap/collision)
- **실패 상황**: 자막 텍스트 1줄이 완전히 포함된 영역의 정가운데 프레임을 분할(Split)할 때 타임스탬프 계산 오류로 인해 자막이 겹치는 현상(Overlap)이나, 시작 시간이 종료 시간보다 뒤로 밀리는 시간 역전(Negative duration) 오류 검증.
- **검증 기준**: 분할 직후 타임코드 충돌 감지기를 실행하여 `start_time >= end_time` 또는 이웃 자막과의 Overlap 발생 여부를 스캔하고, 검출 시 즉각 이전 저널 트랜잭션으로 강제 롤백하고 사용자 경고 팝업을 띄우는지 확인합니다.

#### 4. 60초 숏폼 상한 우회 및 저장소 손상 (Shortform 60s limit bypass & Save/reopen corruption)
- **실패 상황**:
  * 부동소수점 연산 오차로 60.001초 등의 아슬아슬한 임계값 클립이 숏폼 가드를 우회하여 전송을 강제 시도하는 엣지 케이스 검증.
  * 복잡한 연습노트와 메타데이터가 병합된 대형 프로젝트 저장 파일(.json)을 로드할 때, 필드 하나가 누락된 구버전 파일에 대해 파서가 크래시를 일으키며 전체 자막 로드 실패로 이어지는 연쇄 장애 검증.
- **검증 기준**:
  * 숏폼 버짓 계산 시 엄격한 `>= 60.0` float 경계 판정을 수행하는지 검증합니다.
  * 구버전 JSON 세이브 파일 파싱 시 에러 크래시를 방지하는 무해화(Null-safe parsing) 마이그레이션 모듈의 안전 작동을 점검합니다.
