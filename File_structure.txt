[PHASE1-B 리팩토링 요약]

1. 코드 분리 (역할별 Mixin/모듈화)
   - Whisper 백엔드 분리 (OS별: MLX / faster-whisper)
   - MOV 렌더링 분리 (renderer.py)
   - ntfy 알림 분리 (notifier.py)
   - SRT 파서 통합 (srt_parser.py)
   - editor_pipeline → editor_actions 분리
   - main_window → queue_widget / project_ui / cloud_ui 분리

2. 속도 최적화
   - demucs 캐시 재사용 (2회차 ~40초 절약)
   - VAD 캐시 (2회차 ~3초 절약)
   - Whisper 청크 병합 (호출 횟수 30~50% 감소)

3. 멀티 클립 지원 (MVP-1)
   - backend_fast.py: 멀티 클립 배치 전용 파이프라인
   - 자동 프로젝트 생성 (멀티 파일 선택 시)
   - 파일 단위 순차 처리 + SRT 개별 저장

4. 프로젝트 시스템 안정화
   - 프로젝트 JSON 자동 생성/저장
   - 작업 환경(workspace) 저장/복원
   - SRT / JSON 줄바꿈(\u2028) 정규화

5. 상태 관리 개선
   - 완료 판단: backend + state_manager 단일 책임
   - editor_segments에서 완료 판단 제거
   - 큐 테이블 상태 덮어씌우기 버그 해결

6. OS 감지
   - config.py: IS_MAC / IS_WINDOWS / IS_APPLE_SILICON
   - Whisper 백엔드 자동 선택
